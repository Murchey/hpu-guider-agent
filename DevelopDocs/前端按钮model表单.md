# 前端按钮与 AI Prompt Model 对应表

该文档记录了前端各个交互点发送给 AI 时附带的 `model` 标识，用于 AI 智能体识别当前的对话上下文或任务类型。

| 按钮/功能 | Model 标识 | 触发位置 (代码文件) | 功能说明 |
| :--- | :--- | :--- | :--- |
| **问卷画像提交** | `model:用户画像` | `AiDialoguePage.vue` | 提交问卷数据，引导 AI 分析用户偏好 |
| **社交平台发布** | `model:社交内容` | `IndexPage.vue` | 请求 AI 根据方案生成社交媒体文案 |
| **工作模式：互动问答** | `model：智能导游` | `AiDialoguePage.vue` | 默认对话模式，提供通用导游咨询服务 |
| **工作模式：景点介绍** | `model：智能导游` | `AiDialoguePage.vue` | 切换至景点介绍模式，侧重于深度讲解 |
| **单选卡片点击** | `model:景点推荐` | `AiDialoguePage.vue` | 用户在单选弹窗中选择了某个推荐方案 |
| **多选卡片确认** | `model:确定游览方案` | `AiDialoguePage.vue` | 用户在多选弹窗中勾选了感兴趣的景点 |

---

### 注意事项
1. **标识格式**：目前代码中混用了英文冒号 `:` 和中文冒号 `：`（如 `model：智能导游`），在配置智能体解析时建议同时兼容两种符号。
2. **隐藏发送**：除了“工作模式”下的消息，其他带有 `model` 的 Prompt 大多为“隐藏发送”，不会在用户的聊天气泡中显示。
3. **扩展建议**：如果后续增加新的交互弹窗或功能模块，请同步在此表中记录对应的 Model 标识。
