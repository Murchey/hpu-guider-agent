# 测试流程示例

## 业务流程节点

### 节点一：用户画像

输入：`model:用户画像`

```json
{
  "travelNumber": "3",
  "travelDays": "3",
  "travelBudget": "6000",
  "travelStyle": "人文深度",
  "travelFocus": ["行程自由度高", "拍照出片友好"],
  "customHabit": ["拒绝早起赶行程"]
}
```

输出：用户画像分析文本 + 操作按钮

```
[SCENE_DATA]{"btn1":"进行景点推荐","btn2":"","btn3":""}[/SCENE_DATA]
```

---

### 节点二：景点推荐

输入：`model:景点推荐`

输出：推荐景点列表 + 多选卡片

```
[SELECT_DATA]{"options":["鼓浪屿风景区","环岛路","沙坡尾艺术西区","八大关风景区","栈桥","小鱼山公园","情侣路","淇澳岛","珠海大剧院"]}[/SELECT_DATA]
```

---

### 节点三：确定游览景点

输入：`model:确定游览景点` + 用户选择的景点列表

输出：景点确认与路线规划引导

---

### 节点四：路线规划

输入：`model:路线规划` + `start_date` / `travel_mode` / `origin`

输出示例（焦作 → 上海，公共交通方案摘要）：

**方案一（高铁）**
- 车次：G1947（焦作 → 上海虹桥）
- 出发：16:54，到达：23:07
- 票价：二等座 ¥509
- 到达后换乘地铁 2 号线至人民广场站，步行 924 米到达目的地

**方案二（普快 + 公交）**
- 车次：K2188（巩义 → 上海）
- 出发：15:34，到达：次日 05:33
- 票价：硬座 ¥138.5
- 途经焦作市内公交换乘至巩义站，到达后换乘地铁 1 号线

输出还包含天气、行李携带指南等附加信息。

---

### 节点五：文化推荐

输入：`model:文化推荐`

输出包含：
- 全程文化交通解说词
- 分段路线解说
- 特色交通推荐（渡轮、观光巴士、水上巴士等）
- 文化美食推荐
- 文化民俗推荐
- 文化禁忌提醒

---

### 节点六：社交内容生成

输入：`model:社交内容 小红书`

输出：
- 标题（种草风格）
- 正文（按 Day 1/2/3 分段，含上/中/下/晚 时间线）
- 图片提示建议
- 标签推荐
