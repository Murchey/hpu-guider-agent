根据我们在 AiDialoguePage.vue 中的代码逻辑，AI 需要发送的 JSON 格式示例 如下：
注意：只有三个按钮能够定义内容（即btn1、btn2、btn3），其他字段可自定义。

### 1. 标准 JSON 格式
这是 [SCENE_DATA] 标签内部应该包含的纯 JSON 数据：

```
{
  "btn1": "生成：深度历史文化方案",
  "btn2": "生成：亲子休闲游玩方案",
  "btn3": "生成：特种兵式打卡方案"
}
```
### 2. AI 回复时的完整示例
在实际对话中，AI 的回复内容应该长这样（前端会自动解析末尾的标签并弹出对话框）：

```
我已经为您筛选出了几个非常适合的景点：龙门石
窟、白马寺和隋唐洛阳城。

您可以根据自己的偏好，点击下方弹窗中的按钮来
生成详细的行程计划。

[SCENE_DATA]{"btn1": "查看深度文化方案
", "btn2": "查看休闲漫步方案", "btn3": "
查看美食打卡方案"}[/SCENE_DATA]
```
### 3. 格式要点说明
- 字段名 ：必须是 btn1 、 btn2 、 btn3 （全部小写）。
- 可选性 ：如果只需要 1 个或 2 个按钮，只需将多余的字段设为空字符串 "" 或者直接不写该字段，前端会自动隐藏多余按钮。
- 引号 ：必须使用 双引号 （JSON 标准规范）。
- 位置 ：建议放在回复内容的最后一行，方便正则解析且不干扰用户阅读。
